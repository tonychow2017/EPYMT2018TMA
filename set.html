Set operations

Explanation

This is a note on the notion of sets. You may want to skip this tutorial if you know a lot about set theory.
As an introductory tutorial, I am not going to make this note too formal. If you want to look at some materials which are more precise, see here (Wikipedia: ZFC).

What is a set?

You can imagine a set as one of the following:

  • a collection (of object(s))
  • a container
  • an aggregate
  • a heap
  • an unordered list
  • a bag

...etc. However, mathematicians aren't going to define the meaning of a set directly. Again, I mentioned in the class that:

  1. You already know what is the intuitive meaning of a set, and;
  2. It is hard to define what a set actually is. If I define a set as a collection, then you may ask what is the definition of a collection. How should the "definition chain" end?

A simpler way is to assume that you understand what a set actually is.

Belongs to

When we have a collection, we want to describe whether an object is in that collection, or isn't.
We use the symbol ∈ to denote the notion of belongs to. For example, suppose S is a set. Now,

  • x∈S means that x is an element of S, or we say x is in S.
  • x∉S describes that x is not an element of S. (The stroke has the same meaning as that inside the symbol ≠).

Constructions

There are several ways to construct a set, each of which has its own pitfalls.

  1. Construct a set by collecting finite object(s). You can use the following notation (called the builder notation):
    S={x,y,...,z} where x,y and z are objects.
    WARNING: while it would be legitimate to collect finite objects together, it is dangerous if you are going to collect infinite objects. For example, it would be incorrect (I will explain this point in class later) if you let

    S={0.1,0.2,0.3,...,0.01,0.02,...}

    and claim that S is the set of all real numbers between 0 and 1.
  2. Construct a set by restriction. Suppose I have a set S.
    We can construct a set, say, T, by collecting element(s) in S satisfying some property, say, p(x), by a restrction

    T={x∈S:p(x)}

    As an example, let

    S={1,3,5}

    . Then

    T={x∈S:x>2}={3,5}

    . The colon : is read as such that. Some people prefer | instead of :. Therefore, you may also write

    T={x∈S | x>2}
    .
    WARNING: You can do so only when you know a set S. You are not allowed to collect arbitrary objects satisfying some properties. Otherwise, the construction may lead to serious consequences. For example, it is wrong to write

    T={x:x>2}
    .

  3. For the time being you can assume that some sets exist. Indeed, you have to construct them and justify their existences later.
    • ℝ: the set of all real numbers
    • ℤ: the set of all integers
    • ℕ: the set of all natural numbers
    • ℚ: the set of all rational numbers
    • ℂ: the set of all complex numbers

A final note: the meaning of natural numbers is ambiguous. First, 0 may or may not be a natural number, depending on the definition of the author. Also, there are many ways to define what natural numbers are.

Example

  • S={x∈ℕ:x<5}={1,2,3,4}
  • I={x∈ℝ:2<x≤3}=(2,3] (Do you know what an interval means?)

Equality and Subset Relation

Meanings

Given two sets A and B, we say that

  • A is a subset of B if every element in A is an element in B.
    Notation: A⊂B.
    We can also say that B is a superset of A.
  • A and B are equal if A is a subset of B and B is a subset of A.
    Notation: A=B.
    In other words, the sets A and B are equal if every element in A is an element in B, and every element in B is an element in A.

Consequence?

If we portray the meaning of equality in this way, we mean that

  1. a set is unordered. That means
    {1,2,3}={3,2,1}.
  2. a set contains no duplicate element. Even if it does, it has no extra meaning. So,
    {1,2,1,1}={1,2}.

Alternative notations

Some people may also use the symbol ⊆ to denote subset relations. If you want to emphasize that the subset is not equal to the superset, use ⊊.

Operations

Union

We can create a larger (actually, not necessarily larger, but you can think in this way) set by collecting the elements in two sets.

Suppose A and B are two sets. We write

A∪B

to denote this set. You can assume that this set must exist. The larger set is called the union of A and B.

Intersection

Given two sets, we can create a smaller (again, not necessarily but usually) set by taking out the elements which are in both sets.

Suppose A and B and two sets. We write

A∩B

to denote the smaller set. The smaller set is called the intersection of A and B.
The intersection must exist, for, we can use the builder notation to construct the intersection

A∩B={x∈A:x∈B}

.

Complement

We can remove some elements from a set by taking complement. Denote

A\B={x∈A:x∉B}

This is a set which contains element(s) in A which are not in B.
